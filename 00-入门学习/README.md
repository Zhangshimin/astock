# 入门学习

适合小白的 A 股量化学习路线，从零基础到能独立做策略分析。

## 学习路线

```
第1步: 学会获取数据 → Ashare（1个文件搞定）
第2步: 理解K线和技术指标 → baostock + 自己画图
第3步: 写第一个选股策略 → efinance + pandas
第4步: 系统学习量化知识 → ai-quant-trade 教程
第5步: 做回测验证策略 → quanteval / EasyQuant
```

## 收录项目

| 项目 | 难度 | 说明 |
|------|------|------|
| [Ashare](./Ashare/) | ⭐ | 1个文件搞定A股实时行情，零依赖 |
| [baostock](./baostock/) | ⭐ | 免费行情数据，无需注册，入门首选 |
