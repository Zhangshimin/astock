# 入门学习

**献给想学炒股但不知从何下手的小白。**

不需要编程基础，不需要懂量化。这里按你的成长路线收录项目。

## 给新手的真心话

1. **不要一上来就搞量化**。先把 K 线、MACD、KDJ、量价关系看懂
2. **不要跟风买**。用工具自己分析，形成自己的判断
3. **先用模拟盘**。真金白银之前，用这些工具跑几个月模拟
4. **学会止损**。所有赚钱的交易系统，第一步都是控制亏损
