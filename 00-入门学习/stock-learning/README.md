## 这是啥？

**股票入门系统学习资料库**。不用装任何东西，直接在 GitHub 上看。

## 涵盖的内容

```
1️⃣ 股票基础知识       → 开户、交易规则、术语
2️⃣ 股票各种指标       → PE、PB、ROE、MACD、KDJ...
3️⃣ K线基础知识        → 阳线阴线、十字星、锤子线...
4️⃣ 筹码理论           → 主力建仓/洗盘/拉升/出货
5️⃣ 江恩理论           → 时间周期、角度线
6️⃣ 散户亏钱的原因     → 看完少踩很多坑
7️⃣ 庄家分析           → 识别主力动向
8️⃣ 趋势投资           → 顺势而为
9️⃣ 主升浪战法         → 抓主升段
🔟 波浪理论           → 数浪
1️⃣1️⃣ 量价理论         → 量在价先
1️⃣2️⃣~1️⃣5️⃣ 通达信指标公式 → 百日地量、庄影追踪等
```

## 怎么学

```
第1周: 看完 1-3（基础知识 + K线）
第2周: 看完 4-6（筹码 + 庄家 + 避坑）
第3周: 看完 7-9（趋势 + 主升浪）
第4周: 看完 10-11（波浪 + 量价）
第5周: 边看边用 stock-assistant 或 go-stock 实践
```
