## 简介

**从入门到实盘的一站式学习平台**。不是工具，而是一套完整的学习路线。

从小白能看懂的基础，到因子挖掘、机器学习、深度学习、强化学习、图网络、高频交易、大模型，全都有。

## 为什么适合小白

- ✅ **阶梯式学习**：从基础 → 策略 → ML → DL → RL → LLM，一步步来
- ✅ **Jupyter Notebook**：边看边跑代码，不用搭环境
- ✅ **有实盘工具**：学完了有辅助操盘工具可以用
- ✅ **论文解读**：每篇都有代码实现

## 学习路径

```
01-基础知识 → 02-传统策略 → 03-因子挖掘
    ↓
04-机器学习 → 05-深度学习 → 06-强化学习
    ↓
07-图网络 → 08-高频交易 → 09-大模型

10-实盘工具
```
