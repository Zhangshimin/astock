# stock-assistant

> 网址: https://github.com/GhostWang122/stock-assistant  
> 语言: Python + React | 需要一点基础但很好用

## 这是啥？

**AI 自主盯盘 + 自主决策 + 自主模拟交易系统。** 作者把自己多年打磨的交易系统做成了产品，让 AI 在盘中执行这套规则。

## 核心亮点

- 🤖 **AI自主盯盘**：不用一直盯着屏幕，AI帮你监控
- 🎯 **双金策略**：作者成功率较高的交易体系
- 💼 **双模拟账户**：两个账户独立跑，对比不同策略
- 📝 **每日自动复盘**：收盘后自动总结，告诉你哪里做得好哪里不好
- 📊 **Web看板**：在浏览器里看持仓、交易记录、AI分析

## 对小白有什么用

**你可以看到一套真实在用的交易系统长什么样：**
- AI 怎么选股
- 什么时候买入/卖出
- 怎么控制风险
- 收盘后怎么复盘

这才是学炒股最快的方式——**看一个成熟系统的完整运作过程**。

## 一句话总结

**看AI怎么帮你盯盘、交易、复盘，学一套完整的交易系统。**
