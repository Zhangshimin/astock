# go-stock

> 网址: https://github.com/ArvinLovegood/go-stock  
> Stars: 6,434 | 语言: Go + Vue | 有 Windows 安装包

## 这是啥？

**装好就能用的股票分析桌面软件。** 不用写代码，不用配环境，下载安装包打开就能用。

支持 A股/港股/美股。内置 DeepSeek AI 分析。

## 小白能用吗？

✅ **不需要装 Python**
✅ **不需要写代码**
✅ **有 Windows 安装包**
✅ **打开就能看 K 线、资金流向、AI 分析**

## 主要功能

| 功能 | 说明 |
|------|------|
| 📊 K线图 | 日/周/月多周期，支持拉伸放大 |
| 🤖 AI分析 | 集成 DeepSeek，输入股票名就问 |
| 💰 资金流向 | 主力资金进出 |
| 📰 AI热点资讯 | 自动抓取新闻并分析 |
| 🎯 AI智能选股 | 帮你筛选候选 |
| 📈 技术指标 | MACD、RSI、KDJ... |
| 🔔 涨跌提醒 | 价格预警推送 |
| 💼 选股策略 | 内置热门选股策略 |

## 怎么用

1. 去 [Releases 页面](https://github.com/ArvinLovegood/go-stock/releases) 下载安装包
2. 安装打开
3. 配置 DeepSeek API Key（没有的话也能用基础功能）
4. 开始看股票

## 一句话总结

**不想折腾代码？装这个，打开就能 AI 分析股票。**
