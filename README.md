# astock

> 精选 A 股分析开源项目索引库

收集、整理互联网上优秀的 A 股（中国证券市场）分析相关开源项目，涵盖数据获取、技术分析、量化策略、回测框架等方向。

## 目录结构

```
astock/
├── 00-数据获取/        # 行情数据、财务数据、交易接口
├── 01-技术分析/        # K线形态、技术指标、图表分析
├── 02-量化策略/        # 量化选股、策略模型
├── 03-回测框架/        # 回测引擎、绩效评估
├── 04-选股工具/        # 选股策略、条件筛选
├── 05-可视化看板/      # 数据可视化、Dashboard
├── 06-综合平台/        # 一站式分析平台
├── 07-机器学习/        # ML/DL 在 A 股的应用
└── resources/          # 数据源、书籍、工具资源汇总
```

## 收录标准

- **语言**: 以 Python 为主，其他语言亦可
- **活跃度**: 优先收录维护活跃、文档完善的项目
- **实用性**: 真实可用的工具或库，非 toy project
- **开源性**: 开源许可明确

## 使用方式

每个子目录下的 README 说明该分类的概览，项目以独立目录存放，内含项目介绍、功能说明、使用方法及链接。
